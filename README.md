# Radio Station App

Modern React Native (Expo) radio app with MediaCP.net integration.

## Quick Start
```
npm install
npm start
```

## Configure
Edit `src/config/station.js` with your stream URL and MediaCP endpoints.

## Structure
- `App.js` - root
- `src/config/` - station config
- `src/context/` - audio player context
- `src/api/` - MediaCP client
- `src/screens/` - all screens
- `src/components/` - reusable UI
- `src/navigation/` - routing
- `src/hooks/` - custom hooks
- `src/utils/` - helpers
- `src/theme/` - colors/design tokens
